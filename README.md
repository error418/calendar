# calendar-template

Generates a Calendar (as SVG) for a given year.

## Usage

	node bin/cmd.js --year 2017

or

	svgcal --year 2017
	
a SVG file is generated to your current directory.
	
## Configuration

You can customize your calendar appearance in the `config.yml`
